# MPDG PWA Hub

Starter repository for Monterey Peninsula Dental Group PWA apps.

## Purpose

This repository will house dental practice web apps, starting with a treatment-planning PWA.

## Initial apps

- Treatment Planner: tooth/surface selection, procedure search, fees, subtotals, alternative plan comparison, and printable treatment plans.

## Planned structure

```text
MPDG-pwa-hub/
├─ index.html
├─ manifest.json
├─ service-worker.js
├─ apps/
│  └─ treatment-planner/
├─ data/
│  ├─ procedures.csv
│  ├─ fees.csv
│  └─ categories.csv
└─ print/
```
